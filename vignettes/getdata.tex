This brief vingette describes how to load the ISRaD data directly from R
using the ISRaD R package.

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Install (optional)
\end{enumerate}

First as always, you must have installed the ISRaD R package, which
provides the function to download the ISRaD data objects. If you already
have the ISRaD package you can skip this step.

\begin{Shaded}
\begin{Highlighting}[]
\CommentTok{# 1) Install and load the 'devtools' and 'rcrossref' packages}
\CommentTok{# 'devtools' allows you to install the ISRaD package in its "beta"/development form}
\CommentTok{# 'rcrossref' is used by our QAQC tool, but sometimes ISRaD won't install without it}
\KeywordTok{install.packages}\NormalTok{(}\StringTok{"devtools"}\NormalTok{)}
\KeywordTok{library}\NormalTok{(devtools) }
\KeywordTok{install.packages}\NormalTok{(}\StringTok{"rcrossref"}\NormalTok{)}
\KeywordTok{library}\NormalTok{(}\StringTok{"rcrossref"}\NormalTok{)}

\CommentTok{# 2) Install package 'ISRaD' from the github repository:}
\NormalTok{devtools}\OperatorTok{::}\KeywordTok{install_github}\NormalTok{(}\StringTok{"International-Soil-Radiocarbon-Database/ISRaD"}\NormalTok{, }\DataTypeTok{ref=}\StringTok{"master"}\NormalTok{)}
\KeywordTok{library}\NormalTok{(ISRaD) }\CommentTok{# load the package}
\end{Highlighting}
\end{Shaded}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\setcounter{enumi}{1}
\tightlist
\item
  Load the data.
\end{enumerate}

The function to load the data is ISRaD.getdata. This function downloads
ISRaD data if you dont already have it. It also loads the data into R in
one step. It accepts three parameters, directory, dataset, extra.\\
\texttt{directory} = where the data is located or where you want to
download the data. \texttt{dataset} = which dataset you want. This can
be the ``full'' dataset or flattened data objects which are either
``flux'',``interstitial'',``incubation'',``fraction'',or ``layer''.
\texttt{extra} = whether or not you want to download the ISRaD\_extra
object.

In the example below, we are loading the standard ISRaD data object and
storing it as a variable called \texttt{mydata}.

\begin{Shaded}
\begin{Highlighting}[]
\NormalTok{mydata<-}\KeywordTok{ISRaD.getdata}\NormalTok{(}\DataTypeTok{directory =} \StringTok{"~/Desktop/"}\NormalTok{, }\DataTypeTok{dataset=}\StringTok{"full"}\NormalTok{, }\DataTypeTok{extra =}\NormalTok{ F)}
\end{Highlighting}
\end{Shaded}
